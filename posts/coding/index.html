An Armstrong Number (also called a Narcissistic Number) is a number that equals the sum of its digits raised to the power of the number of digits.

Example: 153 is an Armstrong number because 1¬≥ + 5¬≥ + 3¬≥ = 1 + 125 + 27 = 153 

n = 153
temp = n
sum = 0
while temp > 0:
    digit = temp % 10
    sum += digit ** 3
    temp //= 10

if sum == n:
    print("Armstrong")
else:
    print("Not Armstrong")
